# Example 1 - A simple C program

``` bash

$ ls
hello.c
$ make hello
cc     hello.c   -o hello
$ ls
hello  hello.c
$ ./hello 
Hello World!
```


Note that there is no Makefile here, but we can use the make program to compile our code.
